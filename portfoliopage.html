VehiCostCaptor

The primary project for my second term programming class was the development of a vehicle quotation calculator. The attached image showcases the user interface, which allows users to select a vehicle, modify options, and compute the total cost inclusive of sales tax. The calculator also provides the functionality to estimate monthly payments if the user opts for vehicle financing. This requires the user to input a down payment, an interest rate, and a loan term. This assignment holds a special place in my heart since despite the numerous challenges and obstacles encountered during its development, I still persevered and found solutions to solve the problem which I found as a rewarding experience. The step-by-step process throughout the term not only helped me overcome hurdles but also made the journey enjoyable. Through trial and error, I learned to create functionalities using C# and the Windows Forms framework for the user interface. This project stands as a testament to my growth and learning in the second term.
